📖 作品概述

细胞远征 是一款以人体免疫系统为题材的 HTML5 2D 横版平台跳跃游戏。

玩家可自由切换 白细胞、血小板、红细胞 三种免疫细胞， 每种细胞拥有基于真实生理功能设计的独特能力。

游戏模拟了人体擦伤感染的完整病理过程——出血期、创面缺口、炎症潮汐、愈合衰减， 让玩家在闯关中亲历一次免疫应答的微观之旅。

⚙️ 核心机制系统

⚡
能量资源管理
全局能量条 — 出血区持续流失 / 潮涌时加速消耗 / 搭桥挥剑扣能量。
低于阈值降速 & 禁用搭桥，玩家始终在战斗消耗与生存续航间博弈。
🌊
炎症潮汐系统
周期性涌动（3 秒潮涌 / 3 秒消退），潮涌时能量消耗 ×3、移速额外减半。
血小板止血可暂停潮汐；越靠近终点，潮汐强度自动衰减。
🧪
四段生理剧情
① 出血期（8 秒持续失血）→ ② 创面缺口（需血小板搭桥）→ ③ 炎症潮汐（周期性恶化）→ ④ 愈合衰减（越近终点越平静）
🦠
差异化敌人设计
葡萄球菌：死亡分裂为 2 个小体 · 链球菌：蓄力 0.7 秒后直线冲刺（蹲下可躲）
Boss 巨型细菌集群：仅白细胞能伤害，全灭敌人才可通关。

🛠️ 技术实现

技术栈

  • 🖥️ 纯原生技术 — HTML5 Canvas + CSS3 + Vanilla JS，零依赖、零框架
  • 📐 模块化架构 — config → entities → levels → game 四层分离
  • ⚡ 固定步长循环 — 60fps 游戏循环，物理帧与渲染帧分离
  • 💾 localStorage 存档 — 关卡进度、星级评价、速通记录本地持久化
  • 🎵 WebAudio 音效 — 程序化生成，无需音频文件

数据亮点

  • 📦 总体积 ~350KB（含图片资源）
  • 🎨 60+ 调色板常量，像素风格渲染
  • 🧩 5 种敌人类型 + Boss + 迷你分裂体
  • 🎒 8 种道具，各有独立效果与视觉
  • 📱 自适应 800×480 画布，跨平台即开即玩
  • 🌐 部署至 Vercel，全球 CDN 加速
